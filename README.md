# Pharmacy Inventory System Backend

A backend Pharmacy Inventory Management System built using Spring Boot and Java 24.
The project includes OCR-based image recognition to extract medicine information from images such as labels, invoices, and prescriptions.

## Features

* Pharmacy inventory management
* Medicine stock tracking
* OCR image recognition
* RESTful APIs
* Database integration
* Java 24 + Spring Boot backend
* Scalable backend architecture

## Technologies Used

* Java 24
* Spring Boot
* Java SDK
* OCR Technology
* REST API
* Maven
* MySQL

## Frontend Integration

This backend can be integrated with any frontend framework or application, including:

* React
* Angular
* Vue
* Flutter
* Android Apps
* Web Dashboards

You can also use AI tools such as Anthropic Claude or Google Gemini to generate and connect frontend interfaces quickly with the backend APIs.

## API Support

The system provides REST APIs that allow frontend applications to:

* Manage medicines
* Update inventory
* Upload images for OCR processing
* Retrieve pharmacy data
* Track stock levels

## Purpose

This project demonstrates backend development, OCR integration, API development, and modern Java enterprise application architecture for real-world pharmacy systems.
